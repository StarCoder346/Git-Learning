# 🧠 Git Essentials — Interactive Cheatsheet

A beginner-friendly, interactive guide to essential Git commands — built as a single-page HTML project with zero dependencies.

🔗 **Live Site:** [git_learning](https://gitlearning-cyan.vercel.app/)


## ✨ Features

- **Visual Git Workflow** — see how files flow from Working Directory → Staging → Local Repo → Remote
- **21 Essential Commands** — organized into 6 categories: Setup, Basics, Branching, Remote, Undo & Inspect
- **Category Filters** — click to focus on one topic at a time
- **Real Examples** — every command includes a copy-paste-ready example
- **Fully Responsive** — works on desktop, tablet & mobile
- **No Dependencies** — single HTML file, just open and learn

---

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/starcoder346/git-essentials.git

# Open in browser
open index.html
```


## 🗂️ Commands Covered

| Category | Commands |
|----------|----------|
| ⚙️ Setup | `git init` · `git clone` · `git config` |
| 📦 Basics | `git status` · `git add` · `git commit` · `git diff` |
| 🌿 Branching | `git branch` · `git checkout` · `git merge` |
| ☁️ Remote | `git push` · `git pull` · `git remote` · `git fetch` |
| ↩️ Undo | `git restore` · `git reset` · `git revert` |
| 🔍 Inspect | `git log` · `git show` · `git stash` |
